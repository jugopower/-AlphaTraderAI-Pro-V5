# Build021.16 教學手順與 KataGo 自動重連版

新增與修正：

## 教學模式
- 擺黑、擺白、黑白交替時，自動在棋子中央顯示 1、2、3…手順
- 黑棋顯示白字，白棋顯示深色字
- 刪除棋子後，後續手順自動重新編號
- 變化圖備份與還原會保留手順
- 匯出教學 SGF 時，手順以 LB 標籤保存
- 保留 A、B、C、△、○、× 與老師評語

## KataGo
- 連線失敗自動重試最多 3 次
- 第二、第三次重試延長等待時間，處理 Render 休眠喚醒
- 顯示「KataGo 重新連線中」
- 斷線時保留目前棋局
- 不使用 Demo／隨機弱棋備援
- 新增「重新連線」按鈕
- 重新連線成功後，如輪到 AI，會繼續落子

已通過 JavaScript 語法檢查。
